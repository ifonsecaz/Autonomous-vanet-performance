El proyecto se divide en dos partes:
 1. AutoModelCar cubre la implementación de un sistema de detección y rastreo sobre el modelo a escala de vehículo autónomo, el          programa está basado en ROS, escrito en C++ y hace uso de OpenCV
 2. Vanets incluye dos simulaciones sobre redes vehiculares, se implemento usando veins un framework que hace uso de los simuladores     sumo y omnet++
